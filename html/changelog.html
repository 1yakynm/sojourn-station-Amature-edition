04 April 2020

Kazkin and Heedless Defiance updated:

  • Added our own lobby screen!
  • The Entertainment Studio has been moved to a central location near the bar, with a stage added.
  • The Sunken Nightclub was moved to an undisclosed location.
  • Adjusted calibers of all firearms - standardized to a few universal sizes.
  • Changed and added some firearm-related sprites.
  • Added some of the modified weaponry to the armory.
  • Improved the equipment available in the Janitor closet, and gave the role cargo access.
  • Fixed some box, backpack, and lipstick sprites not loading properly.
  • Various map fixes including better wiring, piping, tile/object placement, and lighting.
  • Fixed a loadout error relating to crowbars.
  • Fixed an issue with changing colonist access.
  • Cyborgs now spawn properly in the storage closets and can log out via the same machine that spawns them in.
  • Nanopaste had research points reduced to prevent an exploit.
  • Removed some items from the church bioprinter public disk that should not have been there.
